- FILSify project -

Project Overview


The project aims to create a chat app that enables real-time messaging between users. The app features an HTML-based GUI for the frontend, an Apache2 server for the backend, and SQL for data storage. Sockets are used to handle real-time communication, ensuring a seamless user experience. The project integrates a simple interface with reliable backend functionality to enable smooth and instant messaging between users.

Project Objectives


  • Develop a chat app with real-time messaging capabilities
  • Implement an HTML-based GUI for the frontend
  • Use Apache2 server and Bash for the backend
  • Utilize SQL for data storage
  • Integrate sockets for real-time communication
  • Ensure user privacy and data security
